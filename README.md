# 📊 E-Commerce Sales Analytics Dashboard

An interactive and professional E-Commerce Sales Dashboard built using Python, Pandas, SQL, Streamlit and Plotly.

This project analyzes sales performance, customer trends and product insights using real-world sales data.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/hemant101104/ecommerce-sales-dashboard.git
```

Move into project folder:

```bash
cd ecommerce-sales-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
